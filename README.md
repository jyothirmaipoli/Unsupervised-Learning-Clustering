# Customer Segmentation using Unsupervised Learning (Clustering)

## Project Objective
To group customers into meaningful segments using unsupervised learning algorithms, helping businesses understand customer behavior and make better decisions.

## Dataset
Mall Customer Segmentation Dataset  
Columns: CustomerID, Gender, Age, Annual Income, Spending Score

## Algorithms Used
- K-Means Clustering
- Hierarchical Clustering
- (Optional) DBSCAN

## Tools & Libraries
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Project Flow
1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Apply Clustering Algorithms
5. Visualize Clusters
6. Evaluate using Silhouette Score
7. Results & Conclusion

## Project Files
- `clustering.ipynb` → Jupyter Notebook with code
- `dataset/Mall_Customers.csv` → Dataset used
